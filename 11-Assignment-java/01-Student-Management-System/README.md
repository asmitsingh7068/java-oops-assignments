# 🎓 Student Management System (Console Based)

A simple **Console-Based Student Management System** developed in **Java** using Object-Oriented Programming (OOP) concepts. This project is created as a beginner-friendly assignment to practice Java fundamentals.

---

## 📌 Project Overview

The Student Management System allows users to manage student information through a console interface.

This project focuses on understanding the core concepts of Java OOP rather than using databases or frameworks.

---

## ✨ Features

- ➕ Add Student
- 👀 Display Student Details
- ✏️ Update Student Information
- 🚪 Exit Application

---

## 🛠️ Technologies Used

- Java
- VS Code
- Console Application

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/asmitsingh7068/java-oops-assignments.git
```

2. Open the project in VS Code.

3. Compile the Java files

```bash
javac Main.java Student.java
```

4. Run the application

```bash
java Main
```
