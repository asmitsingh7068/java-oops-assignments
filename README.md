# Java OOP Assignments

This repository contains my Java OOP practice and assignments.  
It is organized topic-wise to strengthen my understanding of core Object-Oriented Programming concepts in Java.

## Topics Covered
- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism

## Folder Structure

### 01-classes-and-objects
Basic programs to understand how classes and objects work in Java.

### 02-encapsulation
Examples showing private fields, getters, setters, and data hiding.

### 03-inheritance
Programs demonstrating parent-child relationships and code reuse.

### 04-polymorphism
Examples of runtime polymorphism using method overriding.
